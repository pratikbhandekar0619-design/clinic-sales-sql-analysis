# Clinic Sales SQL Analysis

## Overview
This project demonstrates revenue and collections analysis using MySQL on
anonymized clinic sales data.

## Objectives
- Analyze total revenue and cash collected
- Measure collection efficiency
- Identify pending and overdue invoices
- Perform ageing analysis on outstanding balances
- Create reusable SQL views for reporting

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## Data Privacy
All data used in this project is anonymized. No real patient, clinic, or financial
information is exposed.
