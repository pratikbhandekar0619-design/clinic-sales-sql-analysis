# Clinic Sales SQL Analysis

## Overview
This project analyzes clinic billing and payment data using MySQL to understand
revenue, collections, pending payments, and overdue risk.

The dataset used in this project is anonymized to protect sensitive clinic and
customer information.

## Objectives
- Calculate total revenue and cash collected
- Measure collection efficiency
- Identify unpaid and overdue invoices
- Perform ageing analysis on outstanding balances
- Create reusable SQL views for dashboards

## Tools Used
- MySQL Workbench
- SQL
- CSV (anonymized data)
- GitHub

## Data Privacy
All personal and sensitive information has been anonymized before publishing.
